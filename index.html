The flow of pushing somthing to GitHub

Run all these commands from inside the Terminal or the Git Bash program in Windows.

  • Cloning a repository

    git clone https://github.com/username/repo-name.git

  • Committing your changes

    Make some changes, and add them to the repo.

    git add .
    git commit -a -m "A good commit message."

  • Adding a remote to your repository

    Fork the repository on GitHub, and visit your fork's web page.
    Once there, copy the remote URL in the bottom right corner.
    And now, add it to this repository.

    git remote add myremote copied-url

  • Pushing the code to github

    git push myremote --all
